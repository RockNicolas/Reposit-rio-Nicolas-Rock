Um pouco Sobre a minha Historia

Minha jornada à programação foi bem do nada, desde pequeno sempre sonhei em ser arquiteto. Por conta de ter crescido no convívio de engenheiros, sempre fui vendo como era incrivel. Os projetos de edificações e de arquitetura, comecei a aprender a mexer nas plataformas de projetos como AutoCAD e etc., aos meus 17 anos comecei a pesquisar mais a fundo no meio. De trabalhos de planta baixa, fiz meu primeiro projeto apenas com o básico do AutoCAD e Com a ajuda de minha mãe e de sua auxiliar, fui aprendendo mais e mais, me apaixonei pela a arquitetura, até que descobrir que o mercado estava ruim para a arquitetura, minha mãe. Me fez uma proposta que eu nunca esqueci, ela disse: "Filho, se é o que você quer, não desista. o mundo é enorme e cheio de oportunidades." então deixei a arquitetura de lado.

Agora vamos realmente falar da minha jornada na programação, falei que cresci no meio de engenheiros. Certo? no meio disso tudo, existia um computador bem básico que consistia em minha mãe fazer seus projetos, quando abrir o cmd pela primeira vez, eu fiquei impressionado na possibilidade que uma pessoa tem de criar algo tão genial como um computador, pesquisava sempre que possível de como acessar tais comandos do cmd, quando eu digitei o DIR/S e todos aqueles arquivos na minha frente diante de meus olhos me senti até um hacker, depois que finalmente aprendi a dominar o Prompt de Comando, comecei a tentar ir mais à frente, consegui a hacker a senha do vizinho apenas pegando as conexões próximas e foi incrivel ter feito algo tão extraordinario, pois eu só tinha 14 anos. Logo após aprender isso, fui atrás de aprender a como desligar máquinas apenas pegando o IP da máquina destinada, comecei fazendo ataques de maquinas conhecidas como o da minha mãe, aos 18 anos decidi fazer faculdade de Ciências da Computação. E descobrir que adiar meu sonho de ser arquiteto não foi a pior coisa do mundo, minha mãe sempre me disse "Filho, Deus fecha janelas para abrir portas" Então descobrir que eu vou ser um programador incrível, bem-vindo ao meu mundo.

Serviços Oferecidos

Como programador Front-end ofereço trabalhos de desenvolvimento web ofereço projetos atuais como desing graficos em interfaces graficas incriveis com as linguagens de HTML5, CSS3, JAVASCRIPT, TYPESCRIPT, BOOTSTRAP 5, PHP e MYSQL.

HTML5

HTML5 (Linguagem de Marcação de Hipertexto 5) é o código que você usa para estruturar uma página web e seu conteúdo. Por exemplo, o conteúdo pode ser estruturado em parágrafos, em uma lista com marcadores ou usando imagens e tabelas. Como o título sugere, este artigo fornecerá uma compreensão básica do HTML5 e suas funções. Prazer HTML5.

CSS3

CSS (Folha de Estilo em Cascata 3) é o código que você usa para dar estilo à suas páginas Web. CSS3 básico apresenta tudo que você precisa para começar. Responderemos a perguntas do tipo: Como mudo meus textos para preto, vermelho, verde, etc? Como faço para que meu conteúdo apareça em determinados lugares na tela?. Prazer CSS3.

JAVASCRIPT

JavaScript (às vezes abreviado para JS) é uma linguagem leve, interpretada e baseada em objetos com funções de primeira classe, mais conhecida como a linguagem de script para páginas Web, mas usada também em vários outros ambientes sem browser, tais como node.js, Apache CouchDB e Adobe Acrobat, Facilita Funções para o nosso HTML5. Prazer JAVASCRIPT.

TYPESCRIPT

O typescript nada mais é que uma extensão da linguagem javascript, ou seja, um javascript com poderes a mais. Ele adiciona recursos de tipagem estática, interfaces, classes e outros recursos orientados a objetos para o javascript. Principalmente em aplicações web, desktop e mobile, Ele tem um facilidade e a agilidade usado de maneira certa. Prazer TYPESCRIPT.

BOOTSTRAP

Bootstrap é um framework de código aberto e gratuito para desenvolvimento de sites e aplicativos. Com recursos para configurar estilos e elementos. O Bootstrap é uma ferramenta usada por front-end e full stack, criado pensando e smartphones se adapte a outros dispositivos, como desktops. Prazer BOOTSTRAP.

PYTHON

Python é uma linguagem de programação de alto nível e de uso geral. Ele usa uma abordagem multi-paradigma, o que significa que ele suporta procedural, orientação a objetos, e algumas construções de programação funcional, Ela tem a forma de leitura e interpretação de codigos simples. Prazer PYTHON

PHP

PHP é uma linguagem de programação voltada originalmente para desenvolvimento de sites e aplicações web. Por ser de código aberto e de fácil aprendizado, é uma das mais populares do mundo e ultizada pro empresas grandes como Googleo, Microsoft, Facebook e etc. Prazer PHP

MYSLQ

O MySQL Online é um sistema de gerenciamento de banco de dados relacional (SGBD) de código aberto que utiliza a linguagem SQL (Structured Query Language). Ele é um dos bancos de dados mais conhecidos do mundo e utilizado, com a facilidade que ela propros ao cliente é pratica e simples. Prazer MYSLQ
